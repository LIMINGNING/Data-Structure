## 3.3.1先序后序中序遍历
（1）先序遍历

遍历过程为：
1. 访问根节点
2. 先序遍历其左子树
3. 先序遍历其右子树
```C
void PreOrderTraversal(BinTree BT)
{
    if(BT)
    {
        printf("%d",BT->Data);  /* 先序遍历根节点 */
        PreOrderTraversal(BT->Left);
        PreOrderTraversal(BT->Right);
    }
}
```
（2）中序遍历

遍历过程为：
1. 中序遍历其左子树
2. 访问根节点
3. 中序遍历其右子树
